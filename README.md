# blog

## Structure
```
your-blog-repo/
├── My First Amazing Blog Post/
│   ├── index.html         (The content of your post)
│   └── cover.jpg          (The cover image for this post)
│
├── Another Great Article/
│   ├── index.html
│   └── cover.jpg
│
└── A Third Post/
    ├── index.html
    └── cover.png          (can be .png, .gif, etc.)
```
